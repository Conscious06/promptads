# Competitor Design Analysis

## Overview
Analyzed top advertising and AI tool landing pages to identify design patterns.

## Competitors Analyzed

### 1. Vercel
- **Colors:** White, #00DC82 (green accent)
- **Style:** Minimal, heavy whitespace, bold typography

### 2. Linear
- **Colors:** Near-black (#08090a), gray text (#9c9da1)
- **Style:** Ultra-minimal, subtle borders, premium feel

### 3. Stripe
- **Colors:** #635BFF (purple-blue accent)
- **Style:** Gradient-heavy, sophisticated

### 4. Anthropic (Claude)
- **Colors:** Warm grays, subtle orange/gold accents
- **Style:** Editorial, content-first

## Common Patterns

| Pattern | Usage |
|---------|-------|
| Dark background | 90% of AI/SaaS |
| Single accent color | Focus attention |
| Subtle gradients | Depth without noise |
| Large serif headings | Premium feel |
| Minimal borders | Clean look |
| Generous spacing | Luxury vibe |

## Current PromptAds Issues

1. **Too much gold** - #d4a853 used too frequently
2. **Gradient visual** - Looks generic AI-generated
3. **Grid features** - Common, not distinctive
4. **No brand mark** - Need a logo/symbol

## Recommended Brand Direction

### New Color Palette
- **Background:** Deep black (#030303) - even darker
- **Surface:** Warm black (#0C0C0C) 
- **Border:** Subtle (#1A1A1A)
- **Text Primary:** Off-white (#F5F5F0)
- **Text Secondary:** Muted (#6B6B6B)
- **Accent:** Electric blue (#3B82F6) - Trust, tech, premium
- **Accent Hover:** Lighter blue (#60A5FA)

### Why This Works
- Blue is more "tech" than gold
- Deeper blacks feel more premium
- Blue builds trust (advertising = trust)
- Different from competitors (most use purple/green)

## Design Changes Needed
1. Remove gold accent, replace with blue
2. Remove gradient circle animation
3. Create brand mark/logo
4. Simplify feature grid
5. Add more whitespace
